|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.01.  Instalar Flutter: ---------------------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Descargar 'Flutter': https://docs.flutter.dev/get-started/install/windows/desktop?tab=download.

                    ⦿   Seleccionar 'Download and install / flutter_windows_3.32.4-stable.zip' (O la version más reciente y estable).
                    ⦿   Descargar la carpeta comprimida en cualquier parte de la PC.

            02. Descomprimir la carpeta en 'C:\'.
            
            03. Ingresar a 'C:\flutter\bin'.
            
            04. Copiar la ruta del explorador, es decir, 'C:\flutter\bin'.

            
|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.02.  Configurar las variables de entorno del usuario y del sistema: ------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Presionar la tecla 'Windows' y escribir 'Editar las variables de entorno del sistema' / 'Enter'.
            
            02. Dar clic en 'Variables de entorno…'.

            03. En 'Variables de usuario' y en 'Variables del sistema':

                    ⦿   Dar click en 'Path' / 'Editar'
                    ⦿   Dar click en 'Nuevo'
                    ⦿   Pegar la ruta 'C:\flutter\bin'
                    ⦿   Dar click en 'Aceptar' / 'Aceptar'.


|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.03.  Verificar la instalación de Flutter con el PowerShell: --------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Abrir el PowerShell de Windows como administrador.

            02. Ejecutar el siguiente comando para instalar 'flutter':

                    PS C:\..\..> flutter

            03. Ejecutar el siguiente comando, para obtener más información de flutter:

                    PS C:\..\..> flutter help

            04. Ejecutar el siguiente comando, para mostrar información sobre las herramientas instaladas e identificar si hay alguna dependencia de 
                plataforma que se necesite para completar la configuración:

                    PS C:\..\..> flutter doctor
            
            05. Posibles errores que se pueden presentar:

                    ⦿	No tiene el componente 'cmdline-tools component'. Escribir el siguiente comando

                        PS C:\..\..> flutter doctor --android-licenses

|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.04.  Instalar Visual Studio Code: ----------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Descargar 'Visual Studio Code': https://code.visualstudio.com/download.

            02. Instalar Visual Studio Code.

                    ⦿   En el asistente de instalación dejar marcado:   ☑ 'Agregar PATH…'.                    

            03. Abrir Visual Studio Code.
            
            04. Instalar las siguientes extensiones en Visual Studio Code:                    
                    
                    ⦿   Dart
                    ⦿   Flutter
                    ⦿   Flutter Color
                    ⦿   Flutter Widget Snippets
                    ⦿   Awesome Flutter Snippets
                    ⦿   Material Icon Theme

            05. Ir a 'File / Preferences / Settings' o 'CTRL + ,' y seleccionar la opción 'Format On Save'.

|-- 02.01.  Crear un proyecto: --------------------------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Crear una carpeta de flutter para los proyectos, por ejemplo en 'C:\..\..\Desktop\flutter'.

            02. Ir a 'view / command palette (Ctrl + shift + p) / Flutter: New Project / Application'.
            
            03. Nombrar el proyecto, ejemplo 'app01' / Enter.
            
|--------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 02.02.  Ejecutar el proyecto en el Navegador: -------------------------------------------------------------------------------------------------------|
|--------------------------------------------------------------------------------------------------------------------------------------------------------|
            
            01. Abrir una 'Terminal' en Visual Studio Code: 'Terminal / New Terminal' ó 'Ctrl + Shift + ñ'.
            
            02. Ejecutar el siguiente comando:

                    PS C:\..\..\flutter\app01> flutter run

                    Edge (web)        • edge    • web-javascript • Microsoft Edge 137.0.3296.93

            NOTA:   ⦿   Tambien se puede acceder de las siguientes formas:

                         A.	'Run / Star debbuging'.
                         B.	En la parte superior derecha, con el Icono de punta de flecha hacia la derecha: 'Run'.

                    ⦿   Para cerrar el servidor, se da clic en la terminal y se combinan las teclas 'CTRL + C' y 'S'.

            03. Al abrir el navegador, se presiona la tecla 'F12' y la combinación de teclas 'CTRL + SHIFT + M' para visualizar como celular.
